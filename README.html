APK Fission

This repository consists of APKBUILD files for software packaged by the APK Fission project directly. The following structure obtains:

  • README.html (this file)
  • main/ (free, open-source/libre software)
  • nonfree/ (software with non-free licenses)

Per APK Fission policy, only software which is rejected from the Adélie or Alpine Linux distributions for licensing reasons only are considered for inclusion in non-free. Other software which is not yet packaged, or which may require special considerations (e.g. 'all-in-one' packages as described on APK Fission's policy page), may be found in main.

Neither repository shall conflict with Adélie's or Alpine's user and community repositories.
